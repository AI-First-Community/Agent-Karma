---
title: Commands
description: Every Agent Karma command (Command Palette → "Agent Karma:").
---

| Command | What it does |
|---|---|
| **Start Session** | Begin a session (intent, AI tool, task type). |
| **End Session** | Finalize — git diff, validation checklist, Karma Score + Phal Card. |
| **Toggle Session** | Quick start/end from the status bar. |
| **Log a Validation** *(Add Validation Command)* | Manually record a Test / Build / Lint / Type Check. |
| **Show Dashboard** | Open the dashboard (trend, heatmap, watchlist). |
| **Generate Karma Card** | Create a shareable certificate of your validation practice. |
| **Check Validation Readiness** | Scan the workspace for test/build/lint/type-check setup. |
| **Find Validation Gaps** | Suggest the next step to improve validation setup. |
| **Install / Remove Pre-Commit Nudge** | Add or cleanly remove the opt-in reminder hook. |
| **Export Current Session as JSON / Markdown** | Export metadata only. |
| **Reset Karma History** | Clear sessions + trend, keep settings. |
| **Delete All Local Data** | Wipe everything. |
| **Weekly Reflection** | A weekly validation-consistency check-in. |
| **Why Did My Karma Move?** | Explain what changed since last session. |
| **Open Settings** | VS Code Settings filtered to Agent Karma. |
| **Toggle Ambient Mode** *(experimental)* | Continuous passive capture (no manual Start). |
| **Toggle AI Usage Reading** *(optional)* | Read Claude Code's local usage logs. |

Most are also reachable from the **Agent Karma** view in the Activity Bar.
